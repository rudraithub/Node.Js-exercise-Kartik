# Men's Ranking

Usage:-
-Start the server: npm start
-Use POST requests to add new rankings: POST /mens
-Use GET requests to view existing rankings: GET /mens

Routes
-POST /mens: Add a new ranking.
-GET /mens: View all rankings.
